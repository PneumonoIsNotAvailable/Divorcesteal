# Divorcesteal

[Download on Modrinth](https://modrinth.com/project/5WGMAZnY)

The official mod for Divorcesteal, not affiliated with the Lifesteal SMP.

[**What is "Divorcesteal"?**](https://www.tumblr.com/divorcesteal/790072306185732096/what-is-divorcesteal)

This mod adds mechanics similar to the Lifesteal SMP, where players can take hearts from each other when killing them,
and players with no hearts are "deathbanned".

The system for unbanning players is the Revive Beacon,
which requires 3 hearts and a randomly selected player head in order to activate.

Players must be added to the Participant List to experience lifesteal mechanics and be rolled in Revive Beacons.
If certain players do not want to play competitively, but still want to log on,
they should be excluded from the Participant List.

The mod also adds several Divorcesteal-specific features, such as a custom painting,
and content from the April Fools event.

This mod will be updated/backported to different versions as Divorcesteal itself requires.
When the server moves away from a version, support for that version will be dropped.
